# 크롬 익스텐션 프로젝트

## 설치 방법

1. 의존성 설치
```bash
npm install
```

2. 빌드
```bash
npm run build
```

개발 모드 (자동 재빌드):
```bash
npm run dev
```

## 크롬 익스텐션 로드 방법

1. 크롬 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. `dist` 폴더 선택

## 기능

0. 한글 -> 영어로 말하기를 연습하게 하는 기능 

1. 팝업에서 영어 세팅 
- 주기 설정 
    - 3분, 5분,10분, 30분 .. 등 
- 난이도 설정 
    - 초보 중급 고급 

2. 해당 주기마다 마우스 옆에 팝업 생성 
3초 카운트다운 표시 후 팝업 노출
|-----------------------------|
|(로고)                        |
|     나는 일요일마다 교회를 간다    |
|                             |
|            흔들어서 정답 보기    |
-------------------------------
 

3. 마우스를 흔들면 정답 표시
마우스 좌클릭 = 닫기 
|-----------------------------|
|(로고)                        |
|i go to church every sunday  |
|                             |
|         마우스 좌클릭으로 닫기    |
-------------------------------
